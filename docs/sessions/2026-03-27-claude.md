### Decisions made
- Claude Code stores MCP servers in `~/.claude.json` via `claude mcp add`; `mcp.json` is not the right config file for Claude Code
- File streams for agent/document saves should use `FileShare.ReadWrite` (not `File.Create`) to coexist with the file watcher
- `get_context` is the single polyglot superpower tool — agents call it first to discover all relevant standards for a given language/task
- `orchestrator.agent.md` routes to all 35 agents; it enforces token discipline and parallelization as first-class concerns
- Session files named `YYYY-MM-DD-{agent}.md`; same-day sessions from different agents get separate files (e.g., `2026-03-27-cursor.md` and `2026-03-27-claude.md`)
